[03] engineering_decisions how i think

DEC-01

One product language, many applications

Repeated interface behavior slowed every new product. So: typed Vue primitives, tokens, and composables built once. Each product keeps its own look. The shared layer stays small.

vue · nuxt · typescript · css-tokens

DEC-02

Small services, legible boundaries

Integrations rot when one service accumulates implicit jobs. So: focused Go gateways, PostgreSQL, explicit contracts, predictable failures. More interfaces. Fewer surprises.

go · postgresql · http-contracts

DEC-03

Make delivery reproducible

Infrastructure knowledge dies in shell history. So: versioned OpenTofu, Ansible, containers, secrets tooling, CI. Pays back on every rebuild.

linux · opentofu · ansible · docker · ci/cd
